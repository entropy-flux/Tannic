\doxysubsection{Detailed Description}
Defines reduction operations for tensor expressions. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
This header provides reduction operations for tensor-\/like objects, implemented as expression templates. Currently supports\+:
\begin{DoxyItemize}
\item argmax\+: Index of maximum value along axis optionally keeping the original rank.
\item argmin\+: Index of minimum value along axis optionally keeping the original rank.
\item sum\+: Sum of the values along axis optionally keeping the original rank.
\item mean\+: Mean of the values along axis optionally keeping the original rank.
\end{DoxyItemize}

All reductions are lazy-\/evaluated and maintain proper shape/dtype transformations. Part of the Tannic Tensor Library.
