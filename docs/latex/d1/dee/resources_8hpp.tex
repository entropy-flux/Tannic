\doxysubsection{Detailed Description}
Memory resource management for heterogeneous computing. 

\begin{DoxyAuthor}{Author}
Eric Hermosis ~\newline
 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
Defines the abstract memory architecture\+:
\begin{DoxyItemize}
\item Host\+: The primary execution environment and memory space (eg. CPU)
\item Device\+: Secondary execution environments with separate memory spaces (eg. GPU)
\item Explicit memory management interface for each domain
\end{DoxyItemize}

The model enforces\+:
\begin{DoxyEnumerate}
\item Separate memory address spaces
\item Distinct allocation semantics per domain ~\newline

\item Explicit data transfer requirements
\end{DoxyEnumerate}
