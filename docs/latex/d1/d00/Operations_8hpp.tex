\doxysubsection{Detailed Description}
Defines expression templates tensor operations. 

\begin{DoxyAuthor}{Author}
Eric Cardozo 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
This header defines expression templates used to represent lazy evaluation of unary and binary tensor operations. The system enables composable arithmetic on tensor-\/like objects while deferring actual computation until evaluation (via {\ttfamily forward()}).

Includes\+:
\begin{DoxyItemize}
\item Type promotion utility ({\ttfamily promote})
\item Shape broadcasting logic ({\ttfamily broadcast})
\item Unary and binary operation templates ({\ttfamily Unary}, {\ttfamily Binary})
\item Arithmetic operation structs ({\ttfamily Addition}, {\ttfamily Subtraction}, etc.)
\item Overloaded operators ({\ttfamily +}, {\ttfamily -\/}, {\ttfamily \texorpdfstring{$\ast$}{*}})
\end{DoxyItemize}

Part of the Tannic Tensor Library.
