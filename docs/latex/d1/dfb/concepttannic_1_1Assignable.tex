\doxysubsection{Detailed Description}
Concept for types that can assign values. 

A type {\ttfamily T} satisfies this concept if it provides a method\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void}\ assign(\textcolor{keyword}{const}\ std::byte*\ ptr,\ std::ptrdiff\_t\ offset);}

\end{DoxyCode}


This function must take a pointer to a raw byte buffer and an offset, then assign the corresponding value into the object.

This concept is mainly used internally when populating tensor elements from a memory buffer (e.\+g., in serialization, slicing, or low-\/level tensor initialization), and enables the usage of slices from custom data structures beside the {\ttfamily Tensor} class.
