class tannic\+::expression\+::\+Reduction$<$ Reducer, Operand $>$}
Lazy reduction expression. 

Represents reductions like {\ttfamily sum}, {\ttfamily max}, or {\ttfamily mean} that collapse tensors along an axis. Output shape/dtype are determined by the {\ttfamily Reducer}\textquotesingle{}s rules.
