\doxysubsection{Detailed Description}
A multidimensional, strided tensor data structure. 

The {\ttfamily \doxylink{classtannic_1_1Tensor}{Tensor}} class is the primary data structure in the Tannic \doxylink{classtannic_1_1Tensor}{Tensor} Library. It represents a dynamic typed intention of a computation. It supports slicing, transposition, and expression composition.\hypertarget{classtannic_1_1Tensor_autotoc_md49}{}\doxyparagraph{\texorpdfstring{Evaluation Mode\+:}{Evaluation Mode:}}\label{classtannic_1_1Tensor_autotoc_md49}

\begin{DoxyItemize}
\item Tensors are currently {\bfseries{eager}}\+: expressions assigned to them are immediately evaluated and stored.
\item A lazy {\ttfamily constexpr} evaluation mode is planned for future versions, enabling compile-\/time computational graphs.
\end{DoxyItemize}\hypertarget{classtannic_1_1Tensor_autotoc_md50}{}\doxyparagraph{\texorpdfstring{Memory\+:}{Memory:}}\label{classtannic_1_1Tensor_autotoc_md50}

\begin{DoxyItemize}
\item \doxylink{classtannic_1_1Host}{Host} and \doxylink{classtannic_1_1Device}{Device} (e.\+g., CUDA) allocations are supported using allocators like {\ttfamily Host()} or {\ttfamily Device()}.
\item Memory must be explicitly allocated via {\ttfamily initialize(\+Host())} or initialize(\+Device()) but maybe removed in the future.
\end{DoxyItemize}\hypertarget{classtannic_1_1Tensor_autotoc_md51}{}\doxyparagraph{\texorpdfstring{Example\+:}{Example:}}\label{classtannic_1_1Tensor_autotoc_md51}
