\subsubsection*{template$<$Expression Source, class... Indexes$>$\newline
class tannic\+::expression\+::\+Slice$<$ Source, Indexes $>$}
Expression template representing a tensor slice or subview. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Source} & The expression type from which the slice is taken (e.\+g., a \doxylink{classtannic_1_1Tensor}{Tensor}). \\
\hline
{\em Indexes} & Variadic list of index types (integer or {\ttfamily \doxylink{structtannic_1_1indexing_1_1Range}{indexing\+::\+Range}}).\\
\hline
\end{DoxyTemplParams}
A {\ttfamily \doxylink{classtannic_1_1expression_1_1Slice}{Slice}} stores\+:
\begin{DoxyItemize}
\item The source expression reference.
\item A tuple of index specifications.
\item The computed shape, strides, and offset for the slice.
\end{DoxyItemize}

It supports further slicing through {\ttfamily operator\mbox{[}\mbox{]}}, element assignment, and scalar comparison for rank-\/0 slices.

This class does not own the underlying data; it merely refers to a subset of it.
