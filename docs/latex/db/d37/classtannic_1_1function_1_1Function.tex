\subsubsection*{template$<$Functional Functor, Expression Argument$>$\newline
class tannic\+::function\+::\+Function$<$ Functor, Argument $>$}
Expression template for mathematical function operations. 

Represents a lazily evaluated unary function that takes a functor as argument and maps it to an argument. The actual computation is deferred until {\ttfamily \doxylink{classtannic_1_1function_1_1Function_a4fadaef6215872de7b9ad0aa15071455}{forward()}} is called.
