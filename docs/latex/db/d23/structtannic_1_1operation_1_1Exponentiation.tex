\doxysubsection{Detailed Description}
Binary element-\/wise exponentiation of two tensor expressions. 

Used to build a lazily evaluated {\ttfamily Binary\texorpdfstring{$<$}{<}\doxylink{structtannic_1_1operation_1_1Exponentiation}{Exponentiation}, Base, Exponent\texorpdfstring{$>$}{>}} expression. The result shape is determined via broadcasting rules, and the result type is chosen according to type promotion rules between the base and exponent.

Computation is {\bfseries{deferred}} until {\ttfamily \doxylink{structtannic_1_1operation_1_1Exponentiation_a87920ea96156ec1c169e8c66304a69a5}{forward()}} is called or the expression is assigned to a {\ttfamily \doxylink{classtannic_1_1Tensor}{Tensor}}, meaning it can be composed into larger expression graphs without incurring immediate cost.\hypertarget{structtannic_1_1operation_1_1Exponentiation_autotoc_md31}{}\doxyparagraph{\texorpdfstring{Example\+: Exponentiation}{Example: Exponentiation}}\label{structtannic_1_1operation_1_1Exponentiation_autotoc_md31}
