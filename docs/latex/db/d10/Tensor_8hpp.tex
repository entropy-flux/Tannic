\doxysubsection{Detailed Description}
Core multidimensional tensor class for the Tannic Tensor Library. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
Defines {\ttfamily \doxylink{classtannic_1_1Tensor}{tannic\+::\+Tensor}}, the primary data structure of the library, supporting\+:


\begin{DoxyItemize}
\item {\bfseries{Dynamic dtypes}}\+: No need for templates, data types can be passed to tensors as arguments to simplify serialization and IO operations.
\item {\bfseries{Memory management}}\+: Host(\+CPU)/\+Device(eg. CUDA) allocation support.
\item {\bfseries{Math operators and functions}}\+: Math operators like +,\texorpdfstring{$\ast$}{*},-\/ and matmul supported, for both CPU and CUDA tensors.
\item {\bfseries{Slicing/views}}\+: Python-\/like indexing ({\ttfamily operator\mbox{[}\mbox{]}}, {\ttfamily range}, {\ttfamily transpose}) with negative index.
\item {\bfseries{Eager evaluation}}\+: Eager execution but allowing compile time optimizations like expression fusion using templates.
\item {\bfseries{Strided storage}}\+: Non-\/contiguous layouts via {\ttfamily Shape} and {\ttfamily Strides}. ~\newline

\end{DoxyItemize}
