\doxysubsection{Detailed Description}
Represents the shape (dimensions) of an tensor-\/like expression. 

The {\ttfamily \doxylink{classtannic_1_1Shape}{Shape}} class provides a constexpr-\/friendly abstraction over an expression shape and provides constructors and accessors for tensor shapes, including support for initializer lists, iterators, and iterable containers.

Shapes are limited to a maximum rank of {\ttfamily Shape\+::limit = 8} dimensions. But this limit may be removed in C++26 when std\+::vector become constexpr friendly.\hypertarget{classtannic_1_1Shape_autotoc_md2}{}\doxyparagraph{\texorpdfstring{Example\+:}{Example:}}\label{classtannic_1_1Shape_autotoc_md2}
