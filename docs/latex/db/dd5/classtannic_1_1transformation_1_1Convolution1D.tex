\doxysubsection{Detailed Description}
Expression for 1D convolution operations. 

Implements 1D convolution with\+:
\begin{DoxyItemize}
\item Input and kernel dtype validation
\item Automatic output shape calculation
\item Support for strides and padding 
\end{DoxyItemize}
