\subsubsection*{template$<$Functional Function, Expression Argument$>$\newline
class tannic\+::function\+::\+Functor$<$ Function, Argument $>$}
Expression template for mathematical function operations. 

Represents a lazily evaluated unary functor that takes a function as argument and maps it to an argument. The actual computation is deferred until {\ttfamily \doxylink{classtannic_1_1function_1_1Functor_abeafec4af1aac21a518659913f841142}{forward()}} is called.
