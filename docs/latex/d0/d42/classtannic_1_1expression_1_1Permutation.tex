\doxysubsection{Detailed Description}
\subsubsection*{template$<$Expression Source, Integral ... Indexes$>$\newline
class tannic\+::expression\+::\+Permutation$<$ Source, Indexes $>$}
Expression template for reordering tensor dimensions according to a specified permutation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Source} & The expression or tensor type being permuted. \\
\hline
{\em Indexes} & A parameter pack of integral indices defining the permutation order.\\
\hline
\end{DoxyTemplParams}
The {\ttfamily \doxylink{classtannic_1_1expression_1_1Permutation}{Permutation}} view reorders the dimensions of a tensor according to a user-\/specified sequence of indices. This changes how elements are accessed along each axis, but does not move the underlying data in memory.

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{Tensor\ \mbox{\hyperlink{classtannic_1_1expression_1_1Permutation_a43662c2291a6a72e55a00a02a6352778}{X}}(\mbox{\hyperlink{classtannic_1_1expression_1_1Permutation_a43662c2291a6a72e55a00a02a6352778}{float32}},\ \{2,\ 3,\ 4\});}
\DoxyCodeLine{\textcolor{keyword}{auto}\ \mbox{\hyperlink{classtannic_1_1expression_1_1Permutation_a43662c2291a6a72e55a00a02a6352778}{Y}}\ =\ \mbox{\hyperlink{namespacetannic_1_1expression_a1f1dde31ec629ed3d6b2b979221cf802}{permute}}(\mbox{\hyperlink{classtannic_1_1expression_1_1Permutation_a43662c2291a6a72e55a00a02a6352778}{X}},\ 2,\ 0,\ 1);\ \textcolor{comment}{//\ shape\ becomes\ (4,\ 2,\ 3)}}

\end{DoxyCode}


\begin{DoxyNote}{Note}
The number of indices in the permutation must match the rank of the tensor. Otherwise, an exception is thrown. 
\end{DoxyNote}
