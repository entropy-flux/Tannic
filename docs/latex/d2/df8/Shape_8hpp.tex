\doxysubsection{Detailed Description}
Defines the {\ttfamily \doxylink{classtannic_1_1Shape}{tannic\+::\+Shape}} class for representing tensor dimensions. ~\newline
 

\begin{DoxyAuthor}{Author}
Eric Cardozo 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
This file is part of the Tannic Tensor Library and defines the {\ttfamily Shape} class, a compact, constexpr-\/friendly abstraction for representing the shape (i.\+e., dimensions) of tensors and expression-\/like objects. Shapes are integral to indexing, broadcasting, and tensor manipulation throughout the library.

The shape supports\+:


\begin{DoxyItemize}
\item Compile-\/time construction (constexpr)
\item Convenient initialization from iterators, lists, or variadic arguments
\item Safe indexing, including Python-\/style negative indexing
\end{DoxyItemize}

All expression-\/like objects in the library expose a {\ttfamily Shape} as part of their interface, making this file central to the Tannic design.
