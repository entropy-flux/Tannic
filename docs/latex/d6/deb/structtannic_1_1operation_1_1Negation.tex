\doxysubsection{Detailed Description}
Unary element-\/wise negation of a tensor expression. 

Used to build a lazily evaluated {\ttfamily Unary\texorpdfstring{$<$}{<}\doxylink{structtannic_1_1operation_1_1Negation}{Negation}, Operand\texorpdfstring{$>$}{>}} expression.
