\doxysubsection{Detailed Description}
Expression for 2D convolution operations. 

Implements 2D convolution with\+:
\begin{DoxyItemize}
\item Input and kernel dtype validation
\item Automatic output shape calculation
\item Support for strides and padding in both spatial dimensions 
\end{DoxyItemize}
