\doxysubsection{Detailed Description}
Defines the core protocol for all expression-\/like types in the Tannic Tensor Library. 

This concept specifies the interface required to participate in tensor operations such as broadcasting, slicing, evaluation, and composition. Any type that satisfies this concept is considered a valid expression—this includes tensors, views, lazy expressions, and future compile-\/time tensor constructs.\hypertarget{concepttannic_1_1Expression_autotoc_md22}{}\doxyparagraph{\texorpdfstring{Requirements\+:}{Requirements:}}\label{concepttannic_1_1Expression_autotoc_md22}
A type {\ttfamily T} satisfies {\ttfamily Expression} if it provides\+:


\begin{DoxyItemize}
\item {\ttfamily dtype() -\/\texorpdfstring{$>$}{>} tannic\+::type} — the data type of the expression.
\item {\ttfamily shape() -\/\texorpdfstring{$>$}{>} \doxylink{classtannic_1_1Shape}{tannic\+::\+Shape} const\&} — the shape (dimensions) of the expression.
\item {\ttfamily strides() -\/\texorpdfstring{$>$}{>} \doxylink{classtannic_1_1Strides}{tannic\+::\+Strides} const\&} — the strides associated with the memory layout.
\item {\ttfamily offset() -\/\texorpdfstring{$>$}{>} std\+::ptrdiff\+\_\+t} — the memory offset from the buffer pointer.
\item {\ttfamily forward() -\/\texorpdfstring{$>$}{>} \doxylink{classtannic_1_1Tensor}{tannic\+::\+Tensor}} — the tensor that will express the computation.
\end{DoxyItemize}

The methods {\ttfamily dtype()}, {\ttfamily shape()}, and {\ttfamily strides()} {\bfseries{can be {\ttfamily constexpr}}}. All major components—such as {\ttfamily Tensor}, views, and expression trees—must satisfy this concept. This ensures a consistent interface for operations like {\ttfamily +}, {\ttfamily matmul}, {\ttfamily broadcast}, etc.

The {\ttfamily Expression} concept is designed to be {\bfseries{extensible}}\+: custom user-\/defined types can participate in the tensor system by conforming to this protocol, enabling interoperability with native Tannic expressions.\hypertarget{concepttannic_1_1Expression_autotoc_md23}{}\doxyparagraph{\texorpdfstring{Example\+:}{Example:}}\label{concepttannic_1_1Expression_autotoc_md23}
