\subsubsection*{template$<$Operator Operation, Expression Operand, Expression Cooperand$>$\newline
class tannic\+::operation\+::\+Binary$<$ Operation, Operand, Cooperand $>$}
Expression template for a binary tensor operation. 

Represents a lazily evaluated binary operation between two expressions. The {\ttfamily \doxylink{classtannic_1_1operation_1_1Binary}{Binary}} class performs type promotion, shape broadcasting, and stride generation upon construction to prepare for efficient evaluation.
