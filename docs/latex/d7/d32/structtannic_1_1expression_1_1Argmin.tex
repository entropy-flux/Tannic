\doxysubsection{Detailed Description}
Finds the {\bfseries{indexes of minimum values}} along an axis. 

Identical to {\ttfamily \doxylink{structtannic_1_1expression_1_1Argmax}{Argmax}} but for minima. Output dtype is {\ttfamily int64}.
