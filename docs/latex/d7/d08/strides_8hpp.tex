\doxysubsection{Detailed Description}
Memory layout specification for tensor dimensions in the Tannic Tensor Library. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
Defines {\ttfamily \doxylink{classtannic_1_1Strides}{tannic\+::\+Strides}}, which determines how elements are spaced in memory across tensor dimensions. Key features\+:


\begin{DoxyItemize}
\item {\bfseries{Memory layout control}}\+: Row-\/major (default), column-\/major, or custom striding.
\item {\bfseries{Constexpr support}}\+: Compile-\/time stride computation.
\item {\bfseries{Shape integration}}\+: Auto-\/computes contiguous strides from {\ttfamily Shape}.
\item {\bfseries{Negative indexing}}\+: Python-\/style dimension access ~\newline

\end{DoxyItemize}

All expression-\/like objects in the library expose an {\ttfamily Strides} as part of their interface, making this file central to the Tannic design.
