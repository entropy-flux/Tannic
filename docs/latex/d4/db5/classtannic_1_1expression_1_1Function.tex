\subsubsection*{template$<$Functional Functor, Composable Argument$>$\newline
class tannic\+::expression\+::\+Function$<$ Functor, Argument $>$}
\doxylink{classtannic_1_1expression_1_1Expression}{Expression} template for mathematical function operations. 

Represents a lazily evaluated unary function that takes a functor as argument and maps it to an argument. The actual computation is deferred until {\ttfamily \doxylink{classtannic_1_1expression_1_1Function_aee2ef7326ed027804524ccf12c9d42ce}{forward()}} is called.
