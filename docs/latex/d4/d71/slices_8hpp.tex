\doxysubsection{Detailed Description}
Implements tensor slicing for expression templates in the Tannic Tensor Library. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 ~\newline
 
\end{DoxyDate}
Defines the {\ttfamily Slice} class template, which represents a view or subrange of a tensor without copying its data. It supports chained indexing, assignment, and comparison operations while preserving the underlying expression structure.

Key features\+:
\begin{DoxyItemize}
\item Zero-\/copy view into a tensor or tensor expression.
\item Supports integer indexing and range slicing.
\item Preserves shape, strides, and data type information.
\item Works as part of the Tannic expression template framework.
\end{DoxyItemize}
