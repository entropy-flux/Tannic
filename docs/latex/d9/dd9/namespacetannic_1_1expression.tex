Creates an expanded view of a tensor, broadcasting singleton dimensions. 

This function returns an {\ttfamily \doxylink{classtannic_1_1expression_1_1Expansion}{Expansion}} expression that allows a tensor to be “expanded” along dimensions of size 1 without copying data. \doxylink{classtannic_1_1expression_1_1Expansion}{Expansion} is only allowed along singleton dimensions; other dimensions must match the requested size.


\begin{DoxyTemplParams}{Template Parameters}
{\em Source} & The tensor or expression type to expand. \\
\hline
{\em Sizes} & Integral dimension sizes for the expanded view. \\
\hline
\end{DoxyTemplParams}

\begin{DoxyParams}{Parameters}
{\em source} & The source tensor or expression. \\
\hline
{\em sizes} & The target shape for the expanded view. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
An {\ttfamily \doxylink{classtannic_1_1expression_1_1Expansion}{Expansion}} object representing the broadcasted view.
\end{DoxyReturn}

\begin{DoxyExceptions}{Exceptions}
{\em \doxylink{classtannic_1_1Exception}{Exception}} & if\+:
\begin{DoxyItemize}
\item The number of dimensions does not match the source rank.
\item A non-\/singleton dimension in the source does not match the requested size.
\end{DoxyItemize}\\
\hline
\end{DoxyExceptions}
