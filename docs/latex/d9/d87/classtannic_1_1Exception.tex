\doxysubsection{Detailed Description}
A simple generic exception type for the Tannic \doxylink{classtannic_1_1Tensor}{Tensor} Library. 

This class provides a lightweight generic exception mechanism during for the frontend of the library. ~\newline


This class is a simple generic \doxylink{classtannic_1_1Exception}{Exception} to avoid overengineering during initial development. As the library matures, this may be extended or replaced with more specialized exception types.
