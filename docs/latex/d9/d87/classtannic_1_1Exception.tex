\doxysubsection{Detailed Description}
Base class for recoverable runtime anomalies in the framework. 

The \doxylink{classtannic_1_1Exception}{Exception} class represents conditions that can be thrown and potentially recovered from during normal program execution. Examples include invalid input, file not found, or other recoverable logical issues.

\begin{DoxyNote}{Note}
Exceptions are intended to be caught and handled. 
\end{DoxyNote}
