\doxysubsection{Detailed Description}
Utilities for index normalization and slicing ranges in the Tannic Tensor Library. 

\begin{DoxyAuthor}{Author}
Eric Cardozo 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 ~\newline
 
\end{DoxyDate}
This header defines helper structures and functions for working with tensor indices and slices. The utilities are inspired by Python\textquotesingle{}s negative indexing and range semantics, providing a consistent and safe interface for normalizing indices into valid positive ranges within tensor dimensions.
