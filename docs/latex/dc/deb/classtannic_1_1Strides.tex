\doxysubsection{Detailed Description}
Represents the memory strides associated with a tensor shape. 

{\ttfamily \doxylink{classtannic_1_1Strides}{Strides}} describes how many elements must be skipped in memory to move between elements along each tensor dimension. It is a key part of defining how multidimensional data is stored and accessed in memory.

A {\ttfamily \doxylink{classtannic_1_1Strides}{Strides}} object can be constructed manually (via variadic arguments or iterators), or automatically from a {\ttfamily \doxylink{classtannic_1_1Shape}{Shape}} (default row-\/major/contiguous layout).

Maximum rank is limited to {\ttfamily \doxylink{classtannic_1_1Strides_a11b903b9305e1258a3c5ddd3e3597e9f}{Strides\+::limit} = 8} for constexpr compatibility and static layout. This limit may be removed in C++26 when std\+::vector become constexpr friendly.\hypertarget{classtannic_1_1Strides_autotoc_md11}{}\doxyparagraph{\texorpdfstring{Example (\+Contiguous Layout)\+:}{Example (Contiguous Layout):}}\label{classtannic_1_1Strides_autotoc_md11}
