This header provides lazy-\/evaluated mathematical functions for tensor-\/like objects, implemented as expression templates. All operations are element-\/wise and maintain the original tensor\textquotesingle{}s shape and strides.

Includes the following mathematical functions\+:
\begin{DoxyItemize}
\item Basic functions\+:
\begin{DoxyItemize}
\item log() -\/ Natural logarithm (ln)
\item exp() -\/ Exponential (e\texorpdfstring{$^\wedge$}{\string^}x)
\item sqrt() -\/ Square root (√x)
\item abs() -\/ Absolute value (\texorpdfstring{$\vert$}{|}x\texorpdfstring{$\vert$}{|})
\end{DoxyItemize}
\item Trigonometric functions (radians)\+:
\begin{DoxyItemize}
\item sin() -\/ Sine
\item cos() -\/ Cosine
\item tan() -\/ Tangent
\end{DoxyItemize}
\item Hyperbolic functions\+:
\begin{DoxyItemize}
\item sinh() -\/ Hyperbolic sine
\item cosh() -\/ Hyperbolic cosine
\item tanh() -\/ Hyperbolic tangent
\end{DoxyItemize}
\end{DoxyItemize}

Part of the Tannic Tensor Library.
