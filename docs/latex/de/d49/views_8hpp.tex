\doxysubsection{Detailed Description}
Implements views for tensors in the Tannic Tensor Library. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 ~\newline
 
\end{DoxyDate}
This header defines expression templates for tensors views without copying data. These views operate on the underlying tensor metadata (shape, strides, and offset) to reinterpret how elements are accessed while preserving the original storage.

Example usage\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Tensor\ X(float32,\ \{2,\ 3\});\ X.initialize();}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{//\ View\ from\ (2,\ 3)\ to\ (3,\ 2)}}
\DoxyCodeLine{\textcolor{keyword}{auto}\ Y\ =\ \mbox{\hyperlink{namespacetannic_1_1expression_a5f5b6b37401a67689b7df785f85eefbe}{view}}(X,\ 3,\ 2);}
\DoxyCodeLine{std::cout\ <<\ Y.shape()\ <<\ std::endl;\ \textcolor{comment}{//\ (3,\ 2)}}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{//\ Swap\ the\ first\ and\ second\ dimensions}}
\DoxyCodeLine{\textcolor{keyword}{auto}\ Z\ =\ \mbox{\hyperlink{namespacetannic_1_1expression_ab43b77ed569cfeb4380abc2e6458aa00}{transpose}}(X,\ 0,\ 1);}
\DoxyCodeLine{std::cout\ <<\ Z.shape()\ <<\ std::endl;\ \textcolor{comment}{//\ (3,\ 2)}}
