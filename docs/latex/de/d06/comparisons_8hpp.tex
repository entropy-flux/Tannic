\doxysubsection{Detailed Description}
Defines element-\/wise comparison operations for tensor expressions. 

\begin{DoxyAuthor}{Author}
Eric Hermosis 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025 
\end{DoxyDate}
This header provides {\bfseries{lazy-\/evaluated comparison operators}} for {\ttfamily Tensor} and expression types. All comparisons are element-\/wise and produce boolean tensors of the same shape as the operands. ~\newline


Supported operators\+:
\begin{DoxyItemize}
\item Equality and inequality\+:
\begin{DoxyItemize}
\item {\ttfamily ==} (equal)
\item {\ttfamily !=} (not equal)
\end{DoxyItemize}
\item Relational comparisons\+:
\begin{DoxyItemize}
\item {\ttfamily \texorpdfstring{$<$}{<}} (less than)
\item {\ttfamily \texorpdfstring{$<$}{<}=} (less than or equal)
\item {\ttfamily \texorpdfstring{$>$}{>}} (greater than)
\item {\ttfamily \texorpdfstring{$>$}{>}=} (greater than or equal)
\end{DoxyItemize}
