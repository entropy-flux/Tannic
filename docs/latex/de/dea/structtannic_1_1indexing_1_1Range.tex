\doxysubsection{Detailed Description}
Represents a half-\/open interval {\ttfamily \mbox{[}start, stop)} for slicing. 

The {\ttfamily \doxylink{structtannic_1_1indexing_1_1Range}{Range}} struct mimics Python\textquotesingle{}s slice behavior\+:
\begin{DoxyItemize}
\item {\ttfamily start} is inclusive
\item {\ttfamily stop} is exclusive
\item Negative values are allowed and are interpreted relative to the size of the dimension being indexed.
\end{DoxyItemize}

By default\+:
\begin{DoxyItemize}
\item {\ttfamily start = 0} (beginning of the axis)
\item {\ttfamily stop = -\/1} (interpreted as "{}until the end"{} when normalized)
\end{DoxyItemize}
