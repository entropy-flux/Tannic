Detailed Description

template<class Reducer, Expression Operand>
class tannic::expression::Reduction< Reducer, Operand >

Expression template for reduction operations.

Represents a lazily evaluated reduction operation applied to a tensor expression. Handles proper shape, stride and dtype transformations for the reduced result.

Template Parameters
ReducerReduction operation type (must implement reduce() and forward())
OperandInput expression type satisfying the Expression concept
