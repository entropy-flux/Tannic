Detailed Description

template<Functional Function, Expression Argument>
class tannic::function::Functor< Function, Argument >

Expression template for mathematical function operations.

Represents a lazily evaluated unary functor that takes a function as argument and maps it to an argument. The actual computation is deferred until forward() is called.

Template Parameters
FunctionA mathematical function satisfying the Functional concept @tparam Argument An expression type satisfying theExpression` concept

Returns
The data type of the underlying argument.

Functions in this expression template system are type-preserving — applying them element-wise does not change the scalar type (e.g., applying exp() to a float32 tensor produces float32 output). Therefore, we can directly query and return the argument's dtype.

Returns the strides of the result.

Returns
A const reference to the strides of the underlying argument.

Strides describe memory layout, and for pure element-wise operations they remain identical to those of the input tensor. Returning const& avoids copying and preserves the original stride information.
