Detailed Description

Computes the mean along an axis.

Requires floating-point input (float32/float64).
Use keepdim=true to maintain shape for broadcasting.

Example:

Tensor X = {{1.0, 2.0},
{3.0, 4.0}};
Tensor Y = mean(X, 0);
// Y = [2.0, 3.0] // (1+3)/2=2.0, (2+4)/2=3.0
