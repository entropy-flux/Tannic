\doxysubsection{Detailed Description}
\doxylink{classtannic_1_1expression_1_1Reduction}{Reduction} argmin operation. 

Finds the indices of minimum values along the specified axis. Reduces input dtype to int64 and removes the reduced dimension.

When reducing along an axis (for argmax/argmin), the output tensor loses that dimension. This requires adjusting the strides accordingly\+:


\begin{DoxyEnumerate}
\item For scalar results (rank-\/1 input reduced to rank-\/0)\+:
\begin{DoxyItemize}
\item Returns empty strides (scalar has no strides)
\end{DoxyItemize}
\item For higher-\/rank tensors\+:
\begin{DoxyItemize}
\item Copies all strides except the one corresponding to the reduced axis
\item Maintains the original memory layout for remaining dimensions
\item The last stride is preserved to maintain contiguous access
\end{DoxyItemize}
\end{DoxyEnumerate}
