Detailed Description

template<class Criteria, Expression First, Expression Second>
class tannic::expression::Comparison< Criteria, First, Second >

Expression template for element-wise tensor comparisons.

Represents a lazy comparison between two tensor expressions. The actual boolean tensor is only materialized when assigned to a Tensor.

Template Parameters
CriteriaThe comparison functor (e.g., EQ, LT)
FirstThe left-hand expression type
SecondThe right-hand expression type
